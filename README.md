## This project demonstrates a simple use of an Arduino in controlling LED brightness. The system utilizes a button to activate a potentiometer, which in turn regulates the brightness of LEDs.

- Arduino Board
- LEDs
- Button
- Potentiometer
- Resistors
- Wires
